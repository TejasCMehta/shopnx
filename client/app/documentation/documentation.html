Installation

Just 2 steps to install the application in your local machine.

Step-1: Install NodeJS(http://nodejs.org), MongoDB(http://mongodb.org) Step-2: run the following code in your command prompt or terminal npm install (You may need to add sudo if in a linux machine) bower install node . Open your web browser and point to http://localhost:8080

There are web hosting solutions which offers free hosting for this APP. (e.g. Openshift, Amazon web services, Heroku)

:::::::::::::: Store Front :::::::::::::::::::::

Store

This example store is a ecommerce application to sell mobiles phones of different brands and types. But this web application is capable of selling any type of products, starting from Home Appliances, Grocerry, Clothes, Footwear, Cosmetics, etc...

This view displays all products with facility for searching, filtering, option to add into cart and checkout with PayPal This has also a auto paging facility which activates on scroll (Like most of the advanced shopping websites) Get detailed information about specific products by clicking in on its name or image See a quick view of present quantity in the cart with option to change cart quantity.

Product Search and Filter

This shopping cart featured with a great quality product filter facility with the ability to sort (by price, product name), filter by brands, categories, price (with a price slider)

Product Details

Shows all available details of the products including the Brand it belongs to The category it is in The MRP and the selling price Product image Other components This also allow users to add the product into the cart or remove from cart from here itself

Orders History

Users: All the orders placed by the logged in user is available in this view.

Administrators: This view presents all orders placed by users with the option to change order status and shipping

Shopping Cart

This store is featured with a shopping cart facility which is very easy to use and super fast. Get quick summary of what is there in Cart, modify the cart quantity and directly checkout using Paypal

Accounts Page

Features like Signup / SignIn / Change Password / Logout is integrated into this application already with high level of security. So you no longer need to be worry about implementing all those features into the application

::::::::::::::: Store Administration ::::::::::::::

Only administrators can access this page

Manage Brands

Administrators can add, edit, delete, filter brands into their store from this view

Manage Categories

Categories are presented in Parent-Child manner in this store for better organisation of products Store's navigation bar at top contains all the categories arranged in parent-child fashion. This view provides facility to add both parent and child categories, edit them, re-arrange category association according to their requirement.

Manage Products

This is the main page for administrators to manage products at store. The right sidebar presents all the available products. Get the product details at the left side by clicking on the product at the right sidebar The left sidebar has option to change product name, brand, category in which this product belongs. This left sidebar also contains a module to manage product variants which has facility for Size, MRP, Price and Image for that perticular variant

Manage Users (Customers)

Using this view administrators can add, remove or edit users of their shopping cart

All Features

### Store Front features

  • Single page web app (SPA) created using AngularJS, NodeJS, Express, MongoDB (MEAN)
  • Fastest shop experience
  • Fast Product Search, Filter with AJAX
  • Price slider and multiple brand selector
  • Faster Add to Cart and Product Details
  • Checkout with Paypal Integration
  • Minimal User Registration process
  • Order history and Password Management
  • Facility for Multi level Category
  • Mobile optimized with Bootstrap
  • Instant updates for any changes made across all clients with SocketIO implementation
  • Loads more products on scroll (No paging required)
  • Clean and responsive user interface

### Store Back Office Features

  • Products, Categories, Brand, Order Management from admin panel with easy directives
  • Manage Order and Change Status from admin panel
  • Facility for Multiple product variants (size, color, price, image)
  • User roles - Administrator, User, Guest
  • SEO friendly URLs for each page
  • Secure and quality code - Takes care all single page web app standards
  • Securely built and prevent security attacks
